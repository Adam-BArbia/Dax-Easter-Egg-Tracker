# Icon Placeholder

The icons currently used in this extension are DuckDuckGo easter egg images and are the property of DuckDuckGo. They are included here as temporary placeholders only.

## Icon Sizes

- **dax-16.png** (16×16 pixels) - Small icon for browser toolbar
- **dax-48.png** (48×48 pixels) - Medium icon for menus/settings
- **dax-128.png** (128×128 pixels) - Large icon for add-on pages

## Note

The extension will still function without custom icons, but they won't display properly in the browser interface.
